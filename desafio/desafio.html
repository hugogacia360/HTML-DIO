A internet

voltar

A internet é uma rede mundial de computadores que permite a troca de informações e comunicação entre pessoas em qualquer lugar do mundo. O surgimento da internet envolve uma série de avanços tecnológicos e desenvolvimentos históricos que ocorreram ao longo do século XX.

A ideia de uma rede global de computadores remonta à década de 1960, quando o governo dos Estados Unidos financiou a criação de uma rede de comunicação de dados chamada ARPANET. Essa rede foi desenvolvida para permitir a comunicação entre computadores de pesquisa em diferentes universidades e instituições militares nos Estados Unidos.

Ao longo dos anos 70 e 80, a ARPANET se expandiu para incluir outras instituições de pesquisa e organizações militares em todo o mundo. Outras redes de computadores foram criadas na Europa e em outros lugares, e o padrão TCP/IP (Transmission Control Protocol/Internet Protocol) foi desenvolvido para permitir a comunicação entre redes diferentes.

Na década de 1990, a World Wide Web (WWW) foi criada pelo cientista da computação britânico Tim Berners-Lee. A WWW permitia que usuários de computador navegasse por páginas da web e acessassem informações de uma variedade de fontes, tornando a internet mais acessível e fácil de usar para o público em geral. O surgimento de navegadores da web como o Netscape Navigator e o Internet Explorer tornou a navegação na web mais fácil e intuitiva.

Com o tempo, a internet se tornou uma ferramenta cada vez mais importante para comunicação, comércio, educação, entretenimento e muito mais. A disseminação global da internet e sua crescente importância em nossas vidas cotidianas são marcos importantes na história da tecnologia moderna.

HTML

voltar

HTML é a linguagem de marcação usada para criar páginas da web. A seguir, estão alguns conceitos básicos do HTML que são úteis para iniciantes:

  • Tags HTML: O HTML usa tags para criar estrutura e conteúdo em uma página da web. As tags são cercadas por colchetes angulares "<" e ">" . Por exemplo, a tag html define o início de um documento HTML.

  • Elementos HTML: Os elementos HTML são formados por uma ou mais tags. Eles são usados para descrever a estrutura do conteúdo de uma página da web, como títulos, parágrafos, imagens, links etc.

  • Atributos HTML: Os atributos HTML fornecem informações adicionais sobre elementos HTML. Eles são usados para modificar a forma como um elemento é exibido ou como ele se comporta. Por exemplo, o atributo "src" é usado para especificar a localização de uma imagem.

  • Estrutura básica do HTML: A estrutura básica de uma página HTML inclui a tag html para indicar o início do documento, a tag head para incluir informações sobre a página, como o título, a tag body para incluir o conteúdo da página e a tag footer para incluir informações adicionais sobre a página.

  • Comentários HTML: Os comentários HTML são usados para adicionar notas para os desenvolvedores de uma página. Eles não são exibidos na página da web, mas são úteis para adicionar notas sobre o código.

  • Tags de estilo: As tags de estilo HTML são usadas para definir a aparência visual de uma página. Elas são usadas para definir a cor, tamanho, fonte, alinhamento e outros aspectos visuais de um elemento.

O que é uma tag HTML ?

voltar

As tags HTML são usadas para criar a estrutura e o conteúdo de uma página da web. Elas são elementos marcados por colchetes angulares "<" e ">" que fornecem informações aos navegadores da web sobre como o conteúdo da página deve ser exibido.

    Aqui estão alguns exemplos de tags HTML comuns:

  1. <"html"> - Esta é a tag que define o início de um documento HTML.

  2. <"head"> - Esta tag é usada para incluir informações sobre a página, como o título e meta informações.

  3. <"body"> - Esta tag é usada para incluir o conteúdo da página, como texto, imagens e outros elementos.

  4. <"p"> - Esta tag é usada para definir um parágrafo.

  5. <"img"> - Esta tag é usada para incluir imagens na página.

  6. <"a"> - Esta tag é usada para criar um link para outra página ou recurso.

  7. <"ul"> - Esta tag é usada para criar uma lista não ordenada.

  8. <"ol"> - Esta tag é usada para criar uma lista ordenada.

  9. <"h1>, <"h2">, <"h3">, etc. - Essas tags são usadas para criar títulos e subtítulos em diferentes níveis.

  10. <"div"> - Esta tag é usada para agrupar elementos em uma seção da página.

Essas são apenas alguns exemplos de tags HTML. Existem muitas outras tags que podem ser usadas para criar uma página da web, incluindo tags para formatar texto, criar tabelas, incorporar vídeos e muito mais. O conhecimento dessas tags é fundamental para quem deseja criar e desenvolver páginas da web.
